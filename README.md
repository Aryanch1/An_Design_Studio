# AN Design Studio Website

## Overview

AN Design Studio is a responsive corporate website developed for an architecture and interior design firm based in Hyderabad, India. The website serves as the company's digital presence, showcasing its services, project portfolio, client testimonials, company information, and career opportunities.

The platform is designed to provide a seamless user experience across desktop, tablet, and mobile devices while maintaining a modern and visually appealing interface.

---

## Features

### Company Showcase

* Professional company overview
* Architecture and interior design services
* Brand identity presentation

### Portfolio Management

* Project gallery
* Floor plan displays
* Design showcase sections
* Interactive portfolio pages

### Client Engagement

* Contact forms
* Inquiry submission
* Live chat integration using Tawk.to

### Career Opportunities

* Dedicated careers page
* Company culture information
* Recruitment and hiring details

### Responsive Design

* Mobile-first layout

* Optimized user experience across devices
